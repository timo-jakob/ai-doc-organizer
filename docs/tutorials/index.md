# Tutorials

Learning-oriented, step-by-step introductions to ai-doc-organizer. A tutorial
takes a newcomer by the hand through a complete, safe, reproducible session —
it teaches by doing, not by explaining (concepts belong in
[Explanation](../explanation/index.md), recipes in
[How-to guides](../how-to/index.md)).

- [Getting started](getting-started.md) — install ai-doc-organizer and run it
  for the first time.

Add a new tutorial as its own page and link it here **and** in the `nav:` of
`mkdocs.yml` — the strict docs build fails on any page missing from the nav.
